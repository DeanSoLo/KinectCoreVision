KinectCoreVision

V 1.4

by Patricio Gonzalez Vivo
patriciogonzalezvivo.com
___________________________________________________________________________________

Implementation of CommunityCoreVision ( ccv.nuigroup.com ) for 3D cameras 
Designed for multitouch tables or interactive instalations that works with TUIO ( tuio.org ) protocol

___________________________________________________________________________________
Initial requirements:

- MacOS ( tested on SnowLeopard ) or Linux ( tested on Ubuntu 11.04 )
- Requires an OpenNI Compliant Sensor
- Requires OpenNI software version 1.3.2.1 or higher
- Requires one of the following:
 * SensorKinect Software version 5.0.1
 * PrimeSense Sensor 5.0.3.4
- Requires NITE software version 1.4.0.5 or higher

OSX Users:
- can delete the libs directory ( it´s just need for linux users )
- can delete the KinectCoreVision file without the icon ( the one with the black terminal on it). That´s the linux binary file.

Linux Users:
- can delete the KinectCoreVision.app dir
